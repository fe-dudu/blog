# Vercel 최적화

## Region 설정

상단 Organization(조직명) - Project(프로젝트명) - setting 탭으로 이동한다.  
Side bar - Functions 탭 목록 중 Function region에서 region을 서울로 설정한다.

## vCPU

상단 Organization(조직명) - Project(프로젝트명) - setting 탭으로 이동한다.  
Side bar - Functions 탭 목록 중 Function CPU에서 적절한 vCPU를 선택한다.

## Ignored Build Step (turborepo)

상단 Organization(조직명) - Project(프로젝트명) - setting 탭으로 이동한다.  
Side bar - git 탭 목록 중 ignored Build Step에서 turborepo 관련 변경사항이 있을 때만 빌드하도록 한다.
