# Terminal ASCII art

## patorjk 사이트에서 Font 선택 및 메시지 입력

먼저 [patorjk](https://patorjk.com/software/taag/#p=display&f=Doh&t=All%20Fit) 사이트에서 원하는 Font를 선택하고 메시지를 입력합니다. 텍스트가 아닌 이미지를 ASCII 아트로 변환을 원할 경우 [asciiart](https://www.asciiart.eu/image-to-ascii)에서 변환할 수 있습니다.

## bash 명령어로 실행 가능하도록 변경

다음과 같이 문장 마지막에 `\n` 문자를 추가하여 bash 명령어로 실행 가능하도록 변경합니다.

```bash
echo -e "\
               AAA               lllllll lllllll      FFFFFFFFFFFFFFFFFFFFFF  iiii          tttt\n\
              A:::A              l:::::l l:::::l      F::::::::::::::::::::F i::::i      ttt:::t\n\
             A:::::A             l:::::l l:::::l      F::::::::::::::::::::F  iiii       t:::::t\n\
            A:::::::A            l:::::l l:::::l      FF::::::FFFFFFFFF::::F             t:::::t\n\
           A:::::::::A            l::::l  l::::l        F:::::F       FFFFFFiiiiiiittttttt:::::ttttttt\n\
          A:::::A:::::A           l::::l  l::::l        F:::::F             i:::::it:::::::::::::::::t\n\
         A:::::A A:::::A          l::::l  l::::l        F::::::FFFFFFFFFF    i::::it:::::::::::::::::t\n\
        A:::::A   A:::::A         l::::l  l::::l        F:::::::::::::::F    i::::itttttt:::::::tttttt\n\
       A:::::A     A:::::A        l::::l  l::::l        F:::::::::::::::F    i::::i      t:::::t\n\
      A:::::AAAAAAAAA:::::A       l::::l  l::::l        F::::::FFFFFFFFFF    i::::i      t:::::t\n\
     A:::::::::::::::::::::A      l::::l  l::::l        F:::::F              i::::i      t:::::t\n\
    A:::::AAAAAAAAAAAAA:::::A     l::::l  l::::l        F:::::F              i::::i      t:::::t    tttttt\n\
   A:::::A             A:::::A   l::::::ll::::::l     FF:::::::FF           i::::::i     t::::::tttt:::::t\n\
  A:::::A               A:::::A  l::::::ll::::::l     F::::::::FF           i::::::i     tt::::::::::::::t\n\
 A:::::A                 A:::::A l::::::ll::::::l     F::::::::FF           i::::::i       tt:::::::::::tt\n\
AAAAAAA                   AAAAAAAllllllllllllllll     FFFFFFFFFFF           iiiiiiii         ttttttttttt"
```

## 프로젝트에 start-dev.sh 파일로 저장

프로젝트에 `start-dev.sh` 파일로 저장합니다.

## 개발 서버 실행 시 메시지가 노출되도록 변경

개발 서버 실행 시 메시지가 노출되도록 변경합니다. Next.js 프로젝트라면 아래와 같이 명령어를 변경합니다.

```json
{
  "scripts": {
    "dev": "bash start-dev.sh && next dev",
    ....
  },
  ...
}
```
