# Slack setting

## Figma
```js filename="Slack 채널 대화창"
/invite @figma
/figma subscribe
```

## Jira
1. Jira에서 Profile image 클릭
2. 드롭다운에서 Slack 클릭
3. Slack 설정에서 Slack Jira 연동
```js filename="Slack 채널 대화창"
/invite @Jira
/jira manage
```

## Confluence
1. Confluence에서 Profile image 클릭
2. 드롭다운에서 Slack 클릭
3. Slack 설정에서 Slack Confluence 연동
```js filename="Slack 채널 대화창"
/invite @Confluence
/confluence notify
```

## Github
```js filename="Slack 채널 대화창"
/github subscribe ${org-name}/${repo-name} comments branches commits issues pulls releases deployments reviews
```
